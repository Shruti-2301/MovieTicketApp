# 🎬 Movie Ticket Booking System

A simple DBMS mini project using **HTML, CSS, JavaScript, Python (Flask)** and **SQLite**.

## Features
- Movie CRUD (Add, View, Update, Delete)
- Ticket Booking with dropdown movie selection
- Automatic total amount calculation
- View and Cancel bookings

```powershell
.venv\Scripts\Activate
python app.py
